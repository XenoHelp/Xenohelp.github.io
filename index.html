No Clients will pop up in Xeno, What to do?

Make sure you are using Roblox Player, Don't use the Microsoft Store version or Bloxstrap. Next, make you are in a Roblox game [E.G: A Baseplate] and press the Chain button. You should now be injected into Roblox and ready to execute scripts.

My Script isn't executing, What to do?

Make sure your script works, Try checking the link and checking the F9 Console inside of Roblox to see if there are any errors. If you've verified that the Script works and that there are no errors, try closing xeno to see if you get a little popup in the bottom-right corner of your screen about a connection to a Server. If nothing pops up then try restarting Roblox and Xeno in the order mentioned above.
